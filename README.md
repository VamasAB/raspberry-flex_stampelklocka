# raspberry-flex_stampelklocka
Automatiseringsscript för uppsättning av stämpelklocka

## Installation
Starta upp Raspberryn och följ ordinarie installationanvisningar. Välj att installera Raspbian/Noobs vid eventuell fråga. Kom ihåg att koppla upp enheten trådlöst mot ett öppet WiFi samt att ta bort lösenordet för användaren

* Land: Sverige
* Språk: Svenska
* Tidszon: Stockholm

* Lösenord: <tomt>
* Bekräfta lösenord: <tomt>
